"Users Module"

The Users module allows you to manage all users registered in the system. You can view user details such as name, email, mobile number, company, and status. Data is fetched in real-time from the backend using REST APIs. This section displays users grouped by company, enabling easy navigation and quick access.

"Company Module"

The Company module displays the complete list of companies along with their contact information. When you select a company, you can view all users associated with that company. The details include company name, code, email, mobile number, and contact person. This module helps maintain organized company–user relationships.

"Upload Module"

The Upload module is used to upload and manage files within the system. Uploaded files are processed and stored in the server with secure handling. You can view uploaded file lists, check their update time, and manage them efficiently. This module is directly connected with the backend using Express and Sequelize.
